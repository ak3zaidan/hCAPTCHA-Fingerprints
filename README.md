# hCAPTCHA-Fingerprints

Collection of real hCAPTCHA fingerprints collected using [hCap Farm](https://github.com/ak3zaidan/web-fingerprint-farm)

Note: To pass hCAPTCHA your UA and TLS FP must match your chrome profile, thats why theres 136 and 137 seperate FP's

Keys.json includes some hCAP keys needed to encrypt/decrypt the challenge for previous versions
